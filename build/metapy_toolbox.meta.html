metapy_toolbox.meta module#

Algorithms module

grid_params_metaheuristic(param_grid, algorithm_setup, general_setup)#

Performs a grid search over metaheuristic algorithm parameters.

Parameters:

  • param_grid (Dict[str, list]) – Dictionary containing the grid of parameters to test. Each key is a parameter name and each value is a list of possible values.

  • algorithm_setup (Dict[str, Any]) – Dictionary with the base setup for the metaheuristic algorithm. Will be updated with each combination.

  • general_setup (Dict[str, Any]) – Dictionary with general settings such as number of repetitions, algorithm type, and seed control.

Returns:

  • results (pd.DataFrame): DataFrame with results for all parameter combinations.

  • best_parameter (pd.Series): Row corresponding to the best parameter set (minimum ‘OF BEST’).

Return type:

Tuple[DataFrame, Series]

metaheuristic_optimizer(algorithm_setup, general_setup)#

This function is responsible for the metaheuristic optimization process. It is a general function that calls the specific algorithm functions.

Parameters:

  • algorithm_setup (Dict[str, Any]) – Dictionary with the metaheuristic optimization configuration: - ‘number of iterations’: int, total number of iterations for the optimization process - ‘number of population’: int, size of the population - ‘number of dimensions’: int, number of decision variables - ‘x pop lower limit’: list, lower bounds for the variables - ‘x pop upper limit’: list, upper bounds for the variables - ‘none variable’: any, optional variable (can be None) - ‘objective function’: callable, objective function to be optimized - ‘algorithm parameters’: dict, specific parameters for the selected optimization algorithm

  • general_setup (Dict[str, Any]) – Dictionary with general settings for the optimization process: - ‘number of repetitions’: int, number of repetitions to perform - ‘type code’: str, type of population (‘real code’ or ‘combinatorial code’) - ‘initial pop. seed’: list, list of seeds (use None for randomness) - ‘algorithm’: str, optimization algorithm identifier

Returns:

Tuple with:

  • all_results_per_rep (list): All results for each repetition

  • best_population_per_rep (list): Best population found in each repetition

  • reports (list): Text reports per repetition

  • status_procedure (int): Index of the best repetition

Return type:

Tuple[List[Any] | None, List[Any] | None, List[str] | None, int | None]
