metapy_toolbox.common_library module#

Module has functions that are used in all metaheuristic algorithms

agent_selection(n_population, n, i_pop=False)#

Selects n agents from a population using uniform selection, optionally excluding one agent.

Parameters:

  • n_population (int) – Total number of individuals in the population.

  • n (int) – Number of agents to select.

  • i_pop (int | bool) – Index to exclude from selection. If False, no exclusion.

Returns:

Tuple containing:

  • selected: Array of selected agent indices.

  • report: Textual report of the selection process.

Return type:

Tuple[ndarray, str]

best_values(x_pop, of_pop, fit_pop)#

This function determines the best, best id, worst particle and worst id. It also determines the average value (OF and FIT) of the population.

Parameters:

  • x_pop (List[List[float]]) – Population of design variables.

  • of_pop (List[float]) – Objective function values for each individual.

  • fit_pop (List[float]) – Fitness values for each individual.

Returns:

Tuple containing:

  • best_id: Index of the best individual.

  • worst_id: Index of the worst individual.

  • x_best: Design variables of the best individual.

  • x_worst: Design variables of the worst individual.

  • of_best: Best objective function value.

  • of_worst: Worst objective function value.

  • fit_best: Best fitness value.

  • fit_worst: Worst fitness value.

  • of_avg: Average objective function value.

  • fit_avg: Average fitness value.

Return type:

tuple

check_interval_01(x_i_old, x_lower, x_upper)#

This function checks if a design variable is out of the limits established x_lower and x_upper, and updates the variable if necessary.

Parameters:

  • x_i_old (List[float]) – Current design variables of the i-th agent.

  • x_lower (List[float]) – Lower bounds for the design variables.

  • x_upper (List[float]) – Upper bounds for the design variables.

Returns:

Corrected design variables within specified bounds.

Return type:

List[float]

convert_continuous_discrete(x, discrete_dataset)#

This function converts a continuous variable into a discrete variable according to a discrete dataset.

Parameters:

  • x (List[float]) – Continuous design variables of the i-th agent.

  • discrete_dataset (Dict[str, List[int | float]]) – Dictionary containing discrete mappings (e.g., {‘x_0’: […], ‘x_1’: […]}).

Returns:

List of converted design variables.

Return type:

List[int | float]

fit_value(of_i_value)#

This function calculates the fitness of the i agent.

Parameters:

of_i_value (float) – Objective function value of the i-th agent.

Returns:

Fitness value of the i-th agent.

Return type:

float

id_selection(n_dimensions, n, k_dimension=False)#

This function selects a k dimension from the all dimensions (uniform selection).

Parameters:

  • n_dimensions (int) – Total number of dimensions in the problem.

  • n (int) – Number of dimensions to select.

  • k_dimension (int | bool) – Index to exclude from selection (default is False, meaning no exclusion).

Returns:

Tuple containing:

  • selected: Array of selected dimension indices.

  • report: String report describing the selection process.

Return type:

Tuple[ndarray, str]

initial_pops(n_repetitions, n_population, n_dimensions, x_lower, x_upper, type_pop, seeds)#

This function randomly initializes a population of the metaheuristic algorithm for a given number of repetitions.

Parameters:

  • n_repetitions (int) – Number of repetitions to generate populations.

  • n_population (int) – Number of individuals per population.

  • n_dimensions (int) – Number of dimensions or problem size.

  • x_lower (List[float] | None) – Lower bounds for real-coded variables. Use None for combinatorial problems.

  • x_upper (List[float] | None) – Upper bounds for real-coded variables. Use None for combinatorial problems.

  • type_pop (str) – Type of encoding. Options: ‘real code’ or ‘combinatorial code’.

  • seeds (List[int | None] | None) – List of seeds (one per repetition). Use None for random seed.

Returns:

List of populations for each repetition.

Return type:

List[List[List[float]]]

initial_population_01(n_population, n_dimensions, x_lower, x_upper, seed=None)#

Generates a random population with defined limits. Continuum variables generator.

Parameters:

  • n_population (int) – Number of individuals in the population.

  • n_dimensions (int) – Number of dimensions (design variables).

  • x_lower (List[float]) – Lower bounds for each variable.

  • x_upper (List[float]) – Upper bounds for each variable.

  • seed (int | None) – Random seed. Use None for a random seed.

Returns:

List of individuals, each represented as a list of real values.

Return type:

List[List[float]]

initial_population_02(n_population, n_dimensions, seed=None)#

The function generates a random population. Combinatorial variables generator.

Parameters:

  • n_population (int) – Number of individuals in the population.

  • n_dimensions (int) – Number of elements in each permutation.

  • seed (int | None) – Random seed. Use None for a random seed.

Returns:

List of individuals, each represented as a list of integers (permutation).

Return type:

List[List[int]]

mutation_01_hill_movement(obj_function, x_i_old, x_lower, x_upper, n_dimensions, pdf, cov, none_variable=None)#

This function mutates a solution using a Gaussian or Uniform distribution. Hill Climbing movement.

Parameters:

  • obj_function (Callable) – Objective function defined by the user.

  • x_i_old (List[float]) – Current design variables of the i-th agent.

  • x_lower (List[float]) – Lower bounds for each variable.

  • x_upper (List[float]) – Upper bounds for each variable.

  • n_dimensions (int) – Problem dimensionality.

  • pdf (str) – Probability density function. Options: ‘gaussian’ or ‘uniform’.

  • cov (float) – Coefficient of variation (in %).

  • none_variable (object | None) – Optional variable to pass to the objective function.

Returns:

Tuple containing:

  • x_i_new: Mutated design variables.

  • of_i_new: Objective function value of the new solution.

  • fit_i_new: Fitness value of the new solution.

  • neof: Number of function evaluations (1).

  • report_move: Mutation process report string.

Return type:

Tuple[List[float], float, float, int, str]

mutation_02_chaos_movement(obj_function, x_i_old, of_i_old, fit_i_old, x_lower, x_upper, n_dimensions, alpha, n_tries, iteration, n_iter, none_variable=None)#

This function mutates a solution using a chaotic maps.

Parameters:

  • obj_function (Callable) – User-defined objective function.

  • x_i_old (List[float]) – Current design variables of the i-th agent.

  • of_i_old (float) – Objective function value of the current solution.

  • fit_i_old (float) – Fitness value of the current solution.

  • x_lower (List[float]) – Lower bounds for each variable.

  • x_upper (List[float]) – Upper bounds for each variable.

  • n_dimensions (int) – Number of design variables.

  • alpha (float) – Chaos control parameter (commonly 4 for full chaos).

  • n_tries (int) – Number of attempts to generate better solutions.

  • iteration (int) – Current iteration number.

  • n_iter (int) – Total number of iterations.

  • none_variable (object | None) – Optional input to be passed to the objective function.

Returns:

Tuple containing:

  • x_i_new: Updated design variables.

  • of_i_new: Updated objective function value.

  • fit_i_new: Updated fitness value.

  • neof: Number of objective function evaluations.

  • report_move: Textual report of the mutation process.

Return type:

Tuple[List[float], float, float, int, str]

quasi_oppositional_population_initialization(obj_function, n_pop, n_dimension, initial_pop, x_lower, x_upper, none_variable=None)#

Generates a quasi-oppositional population and selects the fittest individuals for the initial population.

Parameters:

  • obj_function (Callable) – User-defined objective function that evaluates a population matrix.

  • n_pop (int) – Number of individuals to select for the new population.

  • n_dimension (int) – Number of design variables (problem dimensionality).

  • initial_pop (ndarray) – Initial population array (n_pop x n_dimension).

  • x_lower (List[float]) – Lower bounds for each dimension.

  • x_upper (List[float]) – Upper bounds for each dimension.

  • none_variable (object | None) – Optional additional argument passed to the objective function.

Returns:

New population array (n_pop x n_dimension) containing the fittest individuals from the combined set.
