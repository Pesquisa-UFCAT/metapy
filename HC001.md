<h1>HILL CLIMBING 001</h1>

<p align = "justify">
Standard Hill climbing algorithm. Continuous version. The algorithm also presents the results formatted in a Excel spreadsheet.
</p>

<h2>Theory</h2>

<p align = "justify">The Hill Climbing method is an optimization technique used for local searches, which is one of the first methods of stochastic optimization together with simulated annealing. The difference between the HC (Hill Climbing) and the SA (Simulated Annealing) is that in this method, only movements that improve the objective function are accepted.<br><br>

This procedure tries to find the improved neighboring solution from the set of the neighbors using any adopted acceptance rule such as first improvement, best improvement, random walk, or side walk [1].<br><br>

HC is normally employed along with other optimization methods to improve the exploration and exploitation threshold.<br><br>

Below is the algorithm of the HC optimization method:</p>

```python
1:  Input initial parameters (SIGMA)
2:  X = Initial solution
3:  Calculate OF and FIT
4:  for I in range(N_ITER):
5:      X_TEMP = neighbor solution
6:      if f(X_TEMP) <= f(X):
7:         X = X_TEMP
```

<p align = "justify">
In the case of this algorithm, the generation of neighbors is done randomly and follows a normal distribution where the mean is the value of \(\mathbf{X_i}\) itself in the iteration. The value of \(\sigma\) [<code>SIGMA</code>] is the standard deviation informed by the user.<br><br>

[1]	Al-Betar MA. β -Hill climbing: an exploratory local search. Neural Comput & Applic 2017;28:153–68. https://doi.org/10.1007/s00521-016-2328-2.
</p>

```python 
from META_TOOLBOX import HILL_CLIMBING_001

# Input
PARAMETERS = {'SIGMA': 0.10}

SETUP = {
        'N_REP': 10,
        'N_POP': 5,
        'N_ITER': 1000,
        'X_L': [-2, -2, -2],
        'X_U': [2, 2, 2],
        'D': 3,
        'NULL_DIC': None,
        'PARAMETERS': PARAMETERS
        }


# OF statement
def OF_FUNCTION(X, NULL_DIC):
    X_0 = X[0]
    X_1 = X[1]
    X_2 = X[2]
    OF = X_0 ** 2 + X_1 ** 2 + X_2 ** 2
    return OF

# Call function
RESULTS_REP, BEST_REP, AVERAGE_REP, WORST_REP, STATUS_PROCEDURE = HILL_CLIMBING_001(OF_FUNCTION, SETUP)
```
